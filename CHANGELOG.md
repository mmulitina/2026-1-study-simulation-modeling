# Changelog

## [lab5]

### Added
- Построена сеть Петри для задачи обедающих философов (5 философов, 5 вилок)
- Реализована модель захвата и освобождения вилок философами
- Добавлено детектирование состояния взаимной блокировки (deadlock) — когда каждый философ взял одну вилку и ждёт вторую
- Реализовано имитационное моделирование двух типов:
  - Стохастическое моделирование
  - Детерминированное моделирование
- Создан скрипт для генерации анимации работы сети Петри
- Создан скрипт для формирования отчёта по результатам анализа модели
- Добавлена поддержка литературного программирования (литературный код)
- Реализована генерация из литературного кода:
  - Чистого кода
  - Jupyter Notebook
  - Документации в формате Quarto
- Добавлена возможность вычислений для набора параметров модели

### Changed
- Преобразован исходный код в литературный стиль
- Интегрирована документация в формате Quarto в основной отчёт
- Адаптирована модель для работы с параметризированными запусками

### Fixed
- Выявлено и проанализировано состояние deadlock в базовой модели
- Модифицирована сеть Петри для предотвращения deadlock (указано в целях работы)

### Documentation
