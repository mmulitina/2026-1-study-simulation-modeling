# Changelog

Все значимые изменения в этом проекте будут отражены в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-16

### Добавлено
- Лабораторная работа №7: дискретно-событийное моделирование
- Модель M/M/c (система массового обслуживания)
  - Скрипт `scripts/01_mmc_simulation.jl`
  - Сохранение результатов в CSV
  - Построение графиков: гистограммы времени ожидания и обслуживания, сходимость среднего
- Модель Росса (резервирование с ремонтом)
  - Скрипт `scripts/02_ross_simulation.jl`
  - Несколько ремонтников (параметр `num_repair`)
  - Параметрическое сканирование S = [0,1,2,3] и ремонтников = [1,2]
  - Мониторинг загрузки ремонтников и длины очереди
  - График зависимости времени до краха от S
  - Сравнение с аналитическим решением
- Литературное программирование
  - Добавлены комментарии `# #` и `# ##` для Literate.jl

### Изменено
- Код переведён в структуру DrWatson
- Использованы `datadir()`, `plotsdir()` для сохранения результатов

### Удалено
- Удалены старые нерабочие графики динамики (operational, repair_load)

## [0.1.0] - 2026-05-15

### Добавлено
- Начальная структура проекта DrWatson
- Базовая настройка пакетов
